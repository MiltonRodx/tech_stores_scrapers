# Tech Price Scraper

**Tech Price Scraper** is a Python tool that automates the collection of product data from tech store websites. It scrapes product names, prices, availability, and images, and saves the data in CSV format for easy tracking, analysis, or reporting.

## Features

- Scrapes product details including name, price, and availability
- Downloads product images to a local folder
- Saves metadata and image paths to a CSV file
- Handles pagination to scrape multiple pages automatically
- Configurable and reusable for different categories or stores

## Requirements

- Python 3.8+
- Libraries: `requests`, `beautifulsoup4`, `pandas`

pip install -r requirements.txt
